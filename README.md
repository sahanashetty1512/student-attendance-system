Student Attendance Management System

This project is a digital attendance tracking system designed for both students and teachers.
It includes modules for login, dashboards, attendance marking, reporting, and administration.
The system simplifies attendance management, reduces manual errors, and improves accuracy.
 Features

Student Login – Students can log in and view their attendance percentage.

Teacher Login – Teachers can log in and manage attendance.

Attendance Tracking – Daily attendance can be marked quickly and accurately.

Automated Reporting – Generates attendance reports for daily, weekly, monthly, and semester periods.

Easy-to-Use Interface – Simple and user-friendly UI for all users.
